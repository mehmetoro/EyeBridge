**Eye-controlled Communication & Social Platform for Bedridden Patients**

EyeBridge is a **zero-cost**, **open-source** platform that enables bedridden patients to communicate using only their eyes and a standard webcam. No special hardware required.

## Who Is This For?

- 🛌 **Bedridden patients** in intensive care or long-term care
- 🧠 **ALS, MS, SMA** patients with limited mobility
- 🦽 **Paralyzed** individuals
- 👴 **Elderly** with communication difficulties
- 🏥 **Anyone who cannot use hands or speak** but can move their eyes

## Features

- 👁️ **Eye tracking & blink detection** - Control with just your eyes
- ⌨️ **On-screen keyboard** - Type by looking and blinking
- 🌍 **6 languages supported** - English, Turkish, German, Spanish, French, Russian
- 💬 **Auto-translation** - Message anyone in their own language
- 🎮 **Gamified rehabilitation** - Stay motivated with XP, levels, and badges
- 🤝 **Global community** - Connect with patients worldwide
- 🔊 **Text-to-speech** - Let your device speak for you

## Zero Cost. Zero Hardware. Open Source.

- No expensive eye-tracking devices 
- No special sensors or armbands
- Just a computer/tablet with a webcam

## Technology

| Layer | Technology |
|-------|------------|
| Frontend | React + Vite |
| Database | Supabase (PostgreSQL) |
| Hosting | Vercel (Hobby Plan) |
| Translation | LibreTranslate on Hugging Face |
| Eye Tracking | WebGazer.js / MediaPipe |
| TTS | Web Speech API |
